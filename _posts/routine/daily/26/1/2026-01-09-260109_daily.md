## 🔥 직언 (우선순위 질문에 대한 답변)

> "우선순위들을 정해둔 것만으로도 괜찮겠지?"

**NO. 전혀 괜찮지 않습니다.**

당신은 12월에 똑같은 실수를 했습니다. 12월 초 목표: PS 1일 1솔 + 게임 서버 매일 + 졸업과제 12/31 마감. 결과? **22일 회고 중단 + 졸업과제 미완**.

지금 우선순위 5개 = 게임 서버(수십 시간) + Modern C++(수십 시간) + 네트워크(수십 시간) + 언리얼 프로젝트(수십 시간) + 딥다이브 6개 주제(수백 시간) + DirectX(수십 시간) = **최소 500시간 이상**. "4번까지 할 수 있을 것 같다"? **언제까지? 어떻게?**

**구체적 질문 3개:**
1. 졸업과제는 언제 끝내나요? (2월 목표였는데 우선순위에 없음)
2. 게임 서버는 언제 시작해서 언제 끝내나요? (시작/마감 없음)
3. Modern C++은 몇 월부터 시작하나요? (시작 시점 없음)

**우선순위 나열 ≠ 실행 계획**입니다. 과거 증명: 10~11월 = 회고 + 구체적 마감(정처기 11/8, C++ 11/14) = 성공 ✅. 12월 = 목표만 나열 + 마감 없음 = 실패 ⚠️.

**해야 할 것:**
- **2월 = 졸업과제만** (2/28 마감 설정, 주 15시간)
- **3월 = 게임 서버 완료** (3/31 마감, 주 10시간)
- **4~6월 = Modern C++ + 네트워크** (6/30 마감, 주 10시간)
- **7월 이후 = 언리얼 프로젝트 시작** (딥다이브는 프로젝트 중 필요할 때)
- **DirectX = 2027년 또는 취업 후 연기**

"우선순위 4까지 할 수 있을 것 같다" = 느낌입니다. 느낌 ≠ 계획. 12월에 느낌으로 시도했고 실패했습니다. **이번엔 계획으로 하세요. 월별 마감 + 주간 시간 배치 없으면 = 또 무너집니다.**

지금 당신이 해야 할 건 **모의 게임잼 집중**(1/10~12)입니다. 그 다음(1/13)에 앉아서 **2월~12월 월별 목표 + 마감 + 주간 시간**을 적으세요. 그게 없으면 우선순위는 그냥 위시리스트입니다.
