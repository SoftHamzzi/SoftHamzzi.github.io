# 🏗️ 5F
## 🧩 Fact(사실)
> 무슨 일이 있었나?

- 어제 너무 놀았더니, 컨디션도 안좋고 피곤했다.
  - 집에 와서 푹 잤다.
- 공부를 하지 못했다.
  - 피곤해서 잤던 것도 있고, 아랫집에서 인테리어 공사한다고 너무 시끄러웠다.
- 어제 못했던 주간/월간 회고를 정리하였다.
  - 월간 회고의 피드백은 AI 한도로 인해, 화요일에 올라간다.

- 오늘 나는:
  - 순수 공부 시간: `0m`
  - 착석 난이도: `8/10`
  - 시작 에너지 레벨: `3/10`
  - 최종 에너지 레벨: `4/10`
  - 종료 시각: `오후 11시`

- 시작/최종 에너지 레벨 기준은 회고 정리이다.

## 💭 Feeling(느낌)
> 무슨 느낌이 들었나?

- 요즘 공부 시간이 줄어들어서 잘해보고 싶다.

## 💡 Finding(교훈)
> 어떤 인사이트를 얻었나

- 너무 시끄러워서 공부하지 못하는 상황이 생긴다면, 공부 장소를 옮기는 것도 좋다.

## 🎯 Future action(향후 행동)
> 앞으로 무엇을 해야 할까?

- ✍🏻 트렐로 내일의 계획
  - PS 1일 1솔 + 정리를 한다.
  - 게임 서버 프로그래밍 정리를 시작한다.

## 🔁 Feedback(피드백)
> 앞서 정한 향후 행동을 실천해본 뒤, 이에 대해 어떤 피드백을 받았나?

| 5F 단계 | 주제 | 내용 |
|---------|------|------|
| **S (상황)** | 12/1 월요일 현황 | 순수 공부 시간 **0분**. 전날 친구들과 시간 → 피곤 + 아랫집 인테리어 소음. 집에서 푹 잠. 주간/월간 회고 정리 완료 (월간 피드백은 AI 한도로 12/2 예정). 에너지 레벨: 착석 8/10 (매우 어려움), 시작 3/10, 최종 4/10 (회고 기준). 종료 11시. |
| | 12월 시작 = 0시간 | 12월 첫날 공부 0분 = 11/27 목요일 이후 재발. 당시 친구들과 시간 → 금요일 역전. 이번에도 비슷한 패턴 (친구 → 피곤 → 0시간 → 다음 날?). |
| **B (현재 계획)** | 12/2 화요일 계획 | PS 1솔 + 정리 + 게임 서버 프로그래밍 정리 시작. 구체적 시간/분량 명시 없음. |
| **I (영향)** | 🎯 긍정적 요소 | 주간/월간 회고 완료 = 12월 방향 설정 완료. 피곤 + 소음 상황에서 무리하지 않고 휴식 선택 = 번아웃 방지. "공부 장소 옮기기" Finding = 실천 가능한 대안 제시. "잘해보고 싶다" = 의지 유지. |
| | ⚠️ 개선 필요 사항 | 착석 8/10 + 시작 에너지 3/10 = 매우 낮은 컨디션 → 12/2 시작 에너지도 낮을 가능성. 소음 문제 = 지속 가능성 (인테리어 며칠간?). "요즘 공부 시간 줄어듦" 자각 = 11월 4주차 패턴 (9~13h/7일) 지속 우려. |
| | 💡 "전략적 0시간" 가능성 | 11/27 목 0시간 → 11/28 금 역전 성공 사례. 오늘 0시간 = 피곤 + 소음 = 불가항력 + 휴식 필요. 내일 컨디션 회복 → 시작 가능. |
| **N (다음 단계)** | 1. 12/2 현실적 목표 | 피곤 + 소음 다음 날 = 무리한 목표 금지. PS 1솔 (1h) OR 게임 서버 (1h) 중 **하나만** 선택 권장. 시작 에너지 5/10 이하면 → 가벼운 작업 (난이도 낮은 PS or 게임 서버 읽기만). |
| | 2. 소음 대책 | 인테리어 지속 기간 확인 (관리사무소/아랫집). 지속 시 → 도서관/카페/스터디카페 이동. 귀마개/노이즈캔슬링 이어폰 활용. |
| | 3. 에너지 회복 우선 | 오늘 최종 4/10 → 내일 시작 5/10 이하 예상. 무리하지 말고 1~2시간만 투자 → 성취감 + 리듬 회복. 0시간 연속 방지가 목표. |
| **F (후속 조치)** | 12/2 화요일 저녁 | 실제 공부 시간 (0시간 재발? 1h? 2h?), 소음 지속 여부, 에너지 레벨 (회복?), 수요일 계획 조정. |
