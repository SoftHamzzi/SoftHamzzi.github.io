# 📦 8. NoSQL 기초
## 👉🏻 4. MongoDB를 위한 JSON 이해

### 📄 JSON (JavaScript Object Notation)

- 데이터 트리를 **문자열로 표현**한 데이터 파일
- `{}` 구문을 **객체**라 하며, 객체 안에는 `필드 이름(Key), 필드 값(Value)`의 쌍이 들어간다.

```json
{
	"a" : 1,
	"b" : "x",
	"c" : 100,
	"d" : {"x" : 50, "y" : "qp"},
	"e" : [99, 88, 77, {"pi" : 3.14, "r" : 1}]
}
```

- **키**는 문자열이므로 `" "`으로 둘러싼다.
- **값**은 문자열, 숫자, 객체, 배열을 넣을 수 있다.
- 배열 안에 **객체(`{}`)** 를 가질 수도 있다.
