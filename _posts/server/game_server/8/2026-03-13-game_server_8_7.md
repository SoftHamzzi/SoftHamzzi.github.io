# 📦 8. NoSQL 기초
## 👉🏻 7. 성능 분석 기능

### 📌 개요

- 데이터베이스 **과부하, 병목, 막대한 연산**이 발생하는 경우를 위해 사용한다.
- 대부분의 데이터베이스 엔진에서 지원한다.

---

### 🔧 MongoDB 함수

### 성능 분석

```sql
db.coll1.find().explain("executionStats")
```

- `.explain("executionStats")` 구문을 추가로 넣어서 사용한다.

### 인덱스 추가

```sql
db.coll1.ensureIndex({"a":1})
```

- `.ensureIndex()` 함수를 통해 인덱스를 추가한다.

### 인덱스 목록

```sql
db.coll1.GetIndexes()
```

### 인덱스 제거

```sql
db.coll1.DropIndex({"a":1})
db.coll1.DropIndexes()
```

- 이 외에도 다양한 함수들이 있으며, [mongosh 메서드 - MongoDB Docs](https://www.mongodb.com/docs/mongodb-shell/reference/methods/)에서 확인할 수 있다.
