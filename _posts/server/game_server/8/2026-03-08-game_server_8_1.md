# 📦 8. NoSQL 기초
## 👉🏻 1. 관계형 데이터베이스와 NoSQL

### 🗃️ 관계형 데이터베이스 (RDBMS)

- 필드 구조를 바꿔야 할 때마다 **점점 복잡해진다.**
    - 외래 키를 사용하더라도 마찬가지일 수 있다.
- 기존 레코드 개수가 1억 개라면, **1억 개 전체에 필드를 추가해야 한다.**
    - 필드가 `null` 값을 가질 수 있다 하더라도, **장시간 작동을 멈춘다.**
- 레코드는 **리스트나 배열 구조**의 데이터이다.

---

### 🆕 NoSQL

- 레코드가 **트리나 구조체 형태**를 갖출 수 있다.
