# 📦 9. 분산 서버 구조
## 👉🏻 9. 분산 처리를 엄선해야 하는 이유

### ⚠️ 지나친 분산 처리의 문제점

- **네트워크 장비 과부하**로 서비스 장애 가능성
- **디버깅이 까다로움**
    - 원격 디버깅, 로그 출력 분석 필요
    - 분석 도구가 있지만, 쉽진 않음
- 운영체제가 해야 할 일을 증가시켜, **역효과** 가능성
- **클라우드 환경**에서는 클라우드 서버 인스턴스 간 통신 회선의 신뢰성도 문제가 될 수 있음
    - 분산 처리가 많이 되어, 서버 인스턴스 간 역할 의존이 많으면 문제가 생길 수 있음
    - e.g. 레이턴시가 크게 증가하거나, 통신 불능이 되는 경우
