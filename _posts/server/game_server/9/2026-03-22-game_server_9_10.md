# 📦 9. 분산 서버 구조
## 👉🏻 10. 분산 처리 전략

### 🎯 전략 요약

- 분산 처리를 **피할 수 있다면 피하자.**
- **성능 분석**을 통해 분산 처리 지점을 엄선하자.
- **데이터 응집력**을 확인하고, 분산하자.
- 분산 처리 방식 종류
    - **동기 분산 처리**
    - **비동기 분산 처리**
    - **데이터 동기화 기반 로컬 처리**
