# 📦 4. 게임 서버와 클라이언트
## 👉🏻 항목 2: 온라인 게임에서 게임 서버

### ⌨️ MUD(Multi User Dungeon)

**특징:**

- 전화선 모뎀을 통해 텍스트로만 진행하는 MUD 게임도 있었다.
- 당신은 길을 걷다가 쥐 을(를) 만났다.  
당신은 쥐에게 주먹 을(를) 사용하여 3의 대미지를 입혔다.
- 데디케이티드 서버로 운영하였다.

---

- 이제부터 일반적인 게임 서버는 데디케이티드 서버를 지칭한다.
