# 📦 1. 멀티스레딩
## 👉🏻 1. 프로그램과 프로세스 요약

- **프로그램**: 실행 전 상태. 코드와 데이터로 구성된 파일
- **프로세스**: 프로그램이 실행되어 메모리에 적재된 상태. 코드, 데이터, 힙, 스택 영역을 가진다.

### 🧠 메모리 구조
- 코드와 데이터는 프로그램에서 가져옴
- 힙(동적 메모리)과 스택(함수 호출 기록 및 로컬 변수)이 추가됨

### 🔥 멀티프로세싱
- 여러 프로세스가 동시에 실행되는 것
- 예시: 작업 관리자에서 동일한 프로세스가 여러 개 보이는 경우
