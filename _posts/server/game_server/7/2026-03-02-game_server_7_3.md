# 📦 7. 데이터베이스 기초
## 👉🏻 3. 데이터베이스 사용

### 📚 주요 개념

**테이블 (Table):**

- 표 형태의 데이터 집합

**데이터베이스 인스턴스 (Database Instance):**

- 테이블의 집합

**레코드 (Record):**

- 데이터베이스 테이블의 **행**
- 데이터베이스 상에 저장되어 있는 값들의 모임

**필드 (Field):**

- 데이터베이스 테이블의 **열**
- 특정 종류의 데이터를 저장하는 가장 작은 단위의 열
- 예: 고객 테이블에서 이름, 전화번호, 주소 각각이 하나의 필드

---

### 🔤 필드 타입

**주요 데이터 타입:**

- 정수
- 소수
- 문자열
- 날짜/시간
- 그 외 (바이너리, XML, GUID 등)

---

# 🧐 정리

**데이터베이스 구조:**

```
데이터베이스 인스턴스
└── 테이블 (표)
    ├── 레코드 (행) - 데이터의 한 묶음
    └── 필드 (열) - 데이터의 속성
```

**핵심:**

- 테이블 = 표 형태의 데이터
- 레코드 = 한 줄의 데이터
- 필드 = 각 데이터 항목 (타입 지정 가능)
